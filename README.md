Sphere 3d point rotation
========================

Keys:  

A = Disable x rotation  
Z = Enable x rotation  

Q = Disable y rotation  
S = Enable y rotation  

W = Disable z rotation  
X = Enable z rotation  

Up = Add 10 3d points
Down = Remove 10 3d points
